# Obsidian Vault

这个 vault 用来存放当前工作区相关的长期知识、项目约束和待办记录。

## 使用约定

- `00 Inbox.md`：临时记录、待整理信息。
- `10 Projects/`：按项目存放任务背景、决策和状态。
- `20 Knowledge/`：沉淀稳定知识、规范和参考信息。
- `30 Daily/`：按日期记录短期工作日志。
- `90 Agent Memory.md`：给 Codex 用的工作约束和已确认事实。

## 对 Codex 的意义

我后续可以直接读取和更新这些 Markdown 文件，把它们当作当前工作区的本地知识源。
这不等于模型拥有跨会话内建记忆；真正持久化的是这里的文件内容。
